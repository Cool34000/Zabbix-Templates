📊 Zabbix Templates
Ce dépôt contient une collection de templates Zabbix que j’utilise dans différents environnements de production.
La plupart des templates ont pour source de départ un template trouvé quelque part sur Internet, parfois directement sur les templates de la communauté Zabbix.
Ces templates sont testés sur autant de matériels que possible (parfois un seul modèle !)
Ils ne seront pas forcément adaptés au modèle précis que vous cherchez mais seront en tout cas un point de départ !

🧰 Prérequis
Tous les templates ont été exportés sur un serveur Zabbix v7.4
Vous aurez donc besoin de la même version ou une version plus récente pour les importer !

📦 Installation
Téléchargez le template souhaité, importez dans Zabbix (Data collection → Templates → Import) et monitorez !

⚠️ Avertissement
Ces templates sont fournis tels quels, sans aucune garantie.
Ils ont été testés en environnement réel, mais doivent toujours être validés avant mise en production...
